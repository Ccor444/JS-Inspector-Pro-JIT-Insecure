# JS-Inspector-Pro-JIT-Insecure
Um inspetor de código JavaScript moderno e experimental. Analisa código via AST (Acorn) e Regex, fornecendo relatórios detalhados de Classes, Funções, Globais e DOM IDs. Inclui um modo JIT (Just-In-Time) de scan ao vivo, múltiplas opções de execução (iframe, eval, bind unsafe) e ferramentas para análise profunda de segurança e otimização
